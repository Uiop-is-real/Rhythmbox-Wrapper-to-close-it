# Rhythmbox-Wrapper-to-close-it
I had an issue with Rhythmbox, the standard musicplayer in Linux Mint.
So I asked Claude 4.6 Adaptive about it and we made a Wrapper that closes Rhythmbox, when you click for it to be closed.

Imagine sitting there, needing to stop the music and instead of all sane programs, you close it and it keeps playing. But with this confusing mess this is a problem of the past.

If any changes occur, this will probably break and i wont fix it.
